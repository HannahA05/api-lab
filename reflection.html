AI Reflection

What I Built

A dog image gallery app that fetches random dog photos from the Dog CEO API. Users can click to load images or select a specific breed. Built with vanilla JavaScript, Tailwind CSS, and deployed to be Netlify-ready.

Main Challenges

  • API Quirks: The API returns "succes" (typo) for random images but "success" for breed-specific requests
  • Breed Paths: Some breeds have sub-paths like "retriever/golden" - had to verify with actual API calls
  • Framework Issues: Alpine.js had timing problems, so switched to vanilla JavaScript for reliability

Lessons Learned

  • Test against real APIs early - don't assume response formats
  • Sometimes vanilla JS is simpler than frameworks for small projects
  • Error handling and loading states matter for user experience
